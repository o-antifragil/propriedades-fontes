Propriedades de Fontes no CSS

O CSS oferece diversas propriedades para personalizar a tipografia, como:

  1. font-family – Define a fonte do texto, podendo especificar uma lista de fontes para fallback (exemplo: Arial, Helvetica, sans-serif).
  2. font-size – Controla o tamanho do texto (exemplo: 16px, 1.2em, 100%).
  3. font-weight – Determina a espessura da fonte (valores comuns: normal, bold, lighter, bolder ou numéricos de 100 a 900).
  4. font-style – Define o estilo da fonte (valores: normal, italic, oblique).
  5. font-variant – Controla variantes tipográficas, como versaletes (small-caps).
  6. line-height – Ajusta a altura da linha para melhorar a legibilidade.
  7. letter-spacing e word-spacing – Controlam o espaçamento entre letras e palavras.
  8. text-transform – Modifica a capitalização do texto (uppercase, lowercase, capitalize).

Fontes no CSS

O CSS permite o uso de diferentes fontes, que podem ser classificadas em:

  1. Fontes padrão do sistema – São fontes instaladas no computador do usuário, como Arial, Times New Roman, Courier New.
  2. Web Fonts (Fontes da Web) – Importadas de serviços como Google Fonts ou Adobe Fonts.
  3. Fontes personalizadas – Usadas via @font-face, permitindo importar arquivos de fontes hospedados no próprio servidor.
